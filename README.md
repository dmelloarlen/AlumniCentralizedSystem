The frontend of the **Alumni Centralized System** is a premium, responsive web application designed for the next generation of academic networking. Built with **React 19** and **Tailwind CSS 4**, it delivers a buttery-smooth user experience with a professional SaaS aesthetic.

## ✨ Key Highlights

- **🎨 Modern UI/UX:** Glassmorphism, smooth transitions, and a mobile-first design.
- **💬 Real-time Networking:** Instant 1-on-1 and group messaging powered by ZegoCloud.
- **📊 Interactive Analytics:** Beautifully rendered charts for institutional insights.
- **💼 Career Growth:** A dedicated portal for job discovery and professional development.
- **📅 Event Management:** Seamlessly schedule and join virtual meetings and seminars.

## 🛠️ Tech Stack

| Category | Technology |
| :--- | :--- |
| **Core** | React 19, Vite |
| **Styling** | Tailwind CSS 4, Lucide Icons |
| **Real-time** | ZegoCloud ZIM SDK |
| **Logic** | Axios, React Router 7 |
| **Charts** | Chart.js, React Chartjs 2 |
| **Feedback** | React Hot Toast |

## 🚀 Getting Started

### Prerequisites
- **Node.js** (v18.0.0+)
- **npm** (v9.0.0+)

### Installation

1. **Clone & Enter**
   ```bash
   git clone https://github.com/dmelloarlen/AlumniCentralizedSystem
   cd AlumniCentralizedSystem
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```
4. **Configure Environment**
   Create a `.env` file in the root:
   ```env
   VITE_API_BASE_URL = http://localhost:5000 # or YOUR_BACKEND_DEPLOYMENT_URL
   VITE_ZEGO_SERVER_SECERATE = YOUR_SERVER_SECERATE
   VITE_ZEGO_APP_ID = YOUR_APP_ID
   ```

5. **Ignite the development server**
   ```bash
   npm run dev
   ```

6. **Launch**
   Visit `http://localhost:5173` to see the magic happen. ✨

## 📂 Architecture

- `src/components/` 🧩 - Modular, reusable UI building blocks.
- `src/assets/` 🖼️ - Optimized static assets and branding.
- `App.jsx` 🗺️ - The central routing hub and layout engine.
- `main.jsx` 🏁 - The application bootstrapper.
